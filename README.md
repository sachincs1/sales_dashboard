# 📊 Sales Performance Dashboard – Power BI Project

## 📌 Project Overview
This project presents an interactive Sales Performance Dashboard developed using Microsoft Power BI. The dashboard analyzes sales and profit performance across different regions, cities, customer segments, product categories, and time periods. It helps businesses identify trends, monitor performance, and make data-driven decisions.

## 🎯 Objectives
- Analyze sales and profit performance
- Identify top-performing customers and products
- Detect loss-making regions and cities
- Track monthly and yearly sales trends
- Provide actionable business insights

## 🛠️ Tools & Technologies Used
- Microsoft Power BI
- Power Query (Data Transformation)
- DAX (Data Analysis Expressions)
- Data Visualization Techniques

## 📈 Dashboard Features

### ✅ Sales Overview
- Total Sales, Profit, Quantity, and Discount analysis
- Category-wise and city-wise performance
- Year-wise sales comparison

### ✅ Customer & Time Performance
- Top 5 customers by sales
- Sales performance trends
- Moving average sales analysis (4-week & 12-week)

### ✅ Trends & Distribution
- Monthly sales trend analysis
- Product sales distribution
- Customer distribution across regions

### ✅ Sales & Profit Insights
- Profit and sales comparison by city
- Regional performance ranking
- Identification of negative profit regions

## 📊 Key Insights
- Identifies high-performing regions and products
- Highlights cities generating negative profit
- Helps understand seasonal sales patterns
- Supports business decision-making using visual analytics
